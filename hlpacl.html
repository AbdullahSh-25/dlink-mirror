Operation

Select the operation to be performed on the packets.

Packets from
  • LAN side - packets from the LAN side will be filtered.
Are
  • Deny   - Stop the packet and do not route it.
  • Permit - Pass the packet through.

Protocol

Select a protocol to be filtered.
Any one of the following options may be selected for filtering.

  • IP  -all IP protocols will be filtered.

Source IP/Mask address

An idividual source IP address can be designated for filtering.
If all IP addresses must be filtered, leave this box blank.

  • Enter the IP/subnet mask address in the form of XXX.XXX.XXX.XXX.
Example: The IP address is 192.168.1.21 and the net mask 255.255.255.0. The IP address 0.0.0.0 and the net mask 0.0.0.0 is don't care.

Destination IP address

The destination IP address to be filtered is set.
If all IP addresses must be filtered, leave this box blank.

  • Enter the IP/subnet mask address in the form of XXX.XXX.XXX.XXX.
Example: The IP address is 211.95.68.121 and the net mask 255.255.255.0.
The IP address 0.0.0.0 and the net mask 0.0.0.0 is don't care.
